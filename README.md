# Atmos Weather Dashboard

An aesthetic, minimalist weather application built with a deep dark theme and glassmorphism. Atmos provides real-time weather data, hourly forecasts, and 7-day outlooks with a clean, professional interface.

- **Real-time Weather**: Current temperature, conditions, wind speed, humidity, and "feels like" metrics.
- **Hourly Forecast**: Scrollable horizontal visualization of upcoming weather changes.
- **7-Day Forecast**: Detailed daily outlook for the upcoming week.
- **City Search**: Global city search powered by the Open-Meteo Geocoding API.
- **Aesthetic UI**: High-end glassmorphism design with a dark violet ambient theme.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
